AJAX (Asynchronous JavaScript and XML) est une technique pour  créer des pages web rapides et dynamiques. 
<br>Ses sevices: 
<br>-> Lire les données d’un serveur Web - après que la page a chargé
<br>-> Mettre à jour une page Web sans recharger la page
<br>-> Envoyer/recevoir des données à un serveur Web - en arrière-plan
